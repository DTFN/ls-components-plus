## 安装

```shell
npm install @lingshugroup/web-plus
```

## 浏览器要求

现代浏览器

## 补充

大部分组件基于 `element-plus` 的二次开发，个别组件基于其他第三方组件二次开发

`LSChart` 组件基于 `ECharts` 的二次开发
`LSMap` 组件基于高德地图 `AMap` 的二次开发

`node` 版本：18+
